# Glossaire

Ce glossaire définit les termes importants utilisés dans le projet, en particulier ceux qui pourraient prêter à confusion.

| Terme | Définition |
|-------|-------------|
| Étudiant | Utilisateur principal de la plateforme. Il consulte les cours, compare les choix possibles et planifie son parcours universitaire. |
| Cours | Unité d’enseignement identifiée par un code (ex. IFT2255) et associée à un enseignant, un nombre de crédits, un horaire et une description. |
| Crédits | Mesure de la charge de travail d’un cours. Généralement, 1 crédit équivaut à environ 45 heures de travail total. |
| Prérequis | Cours ou conditions qu’un étudiant doit avoir complétés avant de s’inscrire à un autre cours. |
| Co-requis | Cours qui doivent être suivis simultanément pour être admissible à un autre cours. |
| Programme d’études | Ensemble structuré de cours menant à un diplôme. Il définit les cours obligatoires, optionnels et les règles de progression. |
| Département | Unité académique responsable d’un ensemble de cours (ex. Département d’informatique et de recherche opérationnelle). |
| Trimestre | Période académique pendant laquelle les cours sont enseignés (ex. Automne, Hiver, Été). |
| Horaire | Ensemble des plages horaires où un cours est offert lors d’un trimestre donné. |
| Inscription | Action permettant à un étudiant d’ajouter un cours à son cheminement académique, sous réserve des prérequis et des places disponibles. |
| Résultats académiques | Données agrégées décrivant la performance d’une cohorte pour un cours donné (moyenne, nombre d’inscrits, nombre d’échecs). |
| Avis étudiant | Retour d’expérience d’un étudiant sur un cours, incluant la difficulté perçue, la charge de travail et un commentaire éventuel. |
| Charge de travail | Estimation du temps que demande un cours, incluant cours magistraux, laboratoires, travaux et études personnelles. |
| Difficulté perçue | Niveau de difficulté ressenti par les étudiants pour un cours donné. |
| Historique académique | Liste des cours complétés par un étudiant ainsi que leurs résultats, utilisée pour suivre sa progression dans son programme. |
